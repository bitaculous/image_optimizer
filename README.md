[Image Optimizer]
=================

**Make images load faster.**

Prerequisites
-------------

* [Node.js]
* [npm]

Installation
------------

1. Clone the repository

    ```
    $ git clone git@github.com:bitaculous/image-optimizer.git
    ```

2. Change into the new directory

    ```
    $ cd image_optimizer
    ```

3. Install all dependencies with npm

    ```
    $ npm install
    ```
